Image Hotspots for Drupal 7.x
	by ADCI, LLC team - www.adcillc.com
-------------------------------------------------------------------------------

Description:
Hotspots module allows you to mark the different areas of the image by text labels.

Installation:
1) Install Image Hotspots module.
2) Go to your content type manage fields page. It should have at least one Image field to work with hotspots.
3) Click on the "Edit" link for the Image field and select "Use image hotspots" checkbox.
4) Now you can add hotspots to the Image fields on node edit page.
